Тому з моєї ініціативи за допомогою Gemini 3 Pro & DeepSeek v3 була розроблена ця міні програма для обробки файлів снепшотів Behringer Wing.
> **УВАГА!** ⚠ Перед тестуванням даного ПЗ обов'язково зробіть копію вашого оригінального файлу SNAP. Програма знаходиться у ранньому стані розробки. Застосування програми на ваш власний ризик**
Wing Source Shifter — це зручна десктопна утиліта для візуального керування джерелами (Sources) у файлах снепшотів (.snap) цифрового мікшера Behringer WING.

Програма дозволяє швидко змінювати порядок джерел, перейменовувати їх та змінювати кольори за допомогою інтуїтивно зрозумілої матриці, автоматично оновлюючи маршрутизацію каналів.

## 🚀 Як запустити 

Для користувачів Windows (EXE):

1. Перейдіть до розділу [Releases](https://github.com/Yulian-M/wing-snapshot-source-shifter/releases/tag/v0.01) на сторінці цього репозиторію.
2. Завантажте архів із програмою.
3. Розпакуйте архів та запустіть WingSourceShifter.exe.

> **Примітка:** Якщо Windows Defender видає попередження, натисніть "Докладніше" -> "Виконати в будь-якому разі". Це стандартна реакція на непідписані EXE-файли.

## 📖 Як користуватися

- Відкрийте файл: Натисніть кнопку "Відкрити Snapshot..." та виберіть ваш .snap файл.
- Виберіть групу: У випадаючому списку виберіть потрібну групу джерел (наприклад, Local, AES50 A, USB тощо).
- Змініть порядок:

    - Натисніть на джерело, щоб виділити його.
    - Утримуйте Ctrl або Shift для виділення декількох слотів.
    - Просто перетягніть виділені слоти на нове місце.

- Редагуйте: Натисніть правою кнопкою миші на слот, щоб перейменувати його або змінити колір підсвічування.
- Збережіть: Натисніть "Зберегти Snapshot". Новий файл з'явиться в тій же папці, що й оригінал.
